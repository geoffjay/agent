# Issues

Issues in this project use the [\[x\]it!](https://xit.jotaen.net/) syntax. The
following is a summary of the tags that are most relevant to this project.

## Status

- [ ] Open / Ready
- [@] Ongoing / In progress
- [x] Checked / Complete
- [~] Obsolete / Deprecated
- [?] In question / Unclear

## Priority

- [ ] Unprioritized
- [ ] ! Important
- [ ] !! More important
- [ ] !!!!!! Extremely important
- [ ] ...!!! Same as `!!!` but padded for visual alignment
